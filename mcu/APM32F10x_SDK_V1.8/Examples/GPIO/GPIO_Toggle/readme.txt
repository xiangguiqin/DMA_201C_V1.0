&par Example Description

This example describes how to use DOUT for toggling IO.
The IO of LED2 and LED3 is configured to toggle constantly.
The phenomenon of LED2 and LED3 constantly flickered alternately.


&par Directory contents

  - GPIO/GPIO_Toggle/Source/apm32f10x_int.c     Interrupt handlers
  - GPIO/GPIO_Toggle/Source/main.c              Main program

&par IDE environment

  - MDK-ARM V5.36
  - EWARM V8.50.5.26295

&par Hardware and Software environment

  - This example runs on APM32F103 MINI Devices.

&par Example Description

This example shows how to:
  - Configure the PLL (clocked by HSE) as System clock source
  - Output the System clock on MCO pin(PA8)

&par Directory contents

  - RCM/RCM_ClockConfig/src/apm32f10x_int.c     Interrupt handlers
  - RCM/RCM_ClockConfig/src/main.c              Main program

&par IDE environment

  - MDK-ARM V5.36
  - EWARM V8.50.5.26295

&par Hardware and Software environment

  - This example runs on APM32F103 MINI Devices.

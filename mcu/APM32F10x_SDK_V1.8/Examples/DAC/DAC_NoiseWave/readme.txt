&par Example Description

This example shows how to configure the DAC peripheral to generate noise wave.
The waveform can be displayed using an oscilloscope.
using DAC_CHANNEL_1(PA4) to output noise wave.

&par Directory contents

  - DAC/DAC_NoiseWave/src/apm32f10x_int.c     Interrupt handlers
  - DAC/DAC_NoiseWave/src/main.c              Main program

&par IDE environment

  - MDK-ARM V5.36
  - EWARM V8.50.5.26295

&par Hardware and Software environment

  - This example runs on APM32F103 MINI Devices.

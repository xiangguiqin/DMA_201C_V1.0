&par Example Description

The example shows how to use RCT to generate second interrupt.
download program and then reset (power off to power on).
LED2 will blink every second which is generated by RCT interrupt.

&par Directory contents

  - RTC/RTC_Second/src/apm32f10x_it.c     Interrupt handlers
  - RTC/RTC_Second/src/main.c             Main program

&par IDE environment

  - MDK-ARM V5.36
  - EWARM V8.50.5.26295

&par Hardware and Software environment

  - This example runs on APM32F103 MINI Devices.
